# Laravel 5.5 + vue2.5 + iview + pjax = awesome XD

### install
- ``composer install``
- ``cp .env.example .env``
- ``php artisan key:generate``
- ``npm install (need node.js and npm)``
- ``npm run watch (watch)``
- ``php artisan serve``
- ``done enjoy o_ob``
